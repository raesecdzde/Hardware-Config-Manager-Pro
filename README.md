## Installation and Deployment

For the fastest deployment, we recommend downloading the pre-compiled binary directly from the `/bin` directory. This bypasses the need for local compilation dependencies and ensures the utility runs correctly.

### Option 1: Direct Binary Download (Recommended)

1. Navigate to the [`bin/`](../../releases) directory within this repository.
2. Download the latest compiled release: **[System_Config_Utility_v3.2.exe](../../releases)**.
3. Execute the file with **elevated privileges** (Run as Administrator) to allow for necessary system configuration changes.

### Option 2: Build from Source (Advanced Users)

While direct binary download is the fastest method, advanced users may compile the project:
1. Ensure C++ 17 compiler and necessary dependencies (listed in `src/Dependencies.txt`) are installed.
2. Run the build script: `g++ -o hcm-pro src/*.cpp` 
3. Execute the resulting binary: `./hcm-pro.exe`
